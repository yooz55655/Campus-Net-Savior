# 校园网焚诀

## 介绍

在许多校园网络中，网络访问受到各种限制，如 DNS 劫持、流量限制、访问控制等。这个项目的目标是通过使用 OpenWRT 路由器，帮助用户绕过这些限制，提升校园网的使用体验。我们将介绍如何配置和优化 OpenWRT 路由器，使其适应不同的校园网络环境，并提供解决方案来应对常见的网络限制和局限性。

## 目标

本项目提供了一系列教程和配置技巧，帮助你：
- 绕过校园网的 DNS 劫持
- 实现校园网内访问限制解除
- 提升校园网的带宽和稳定性
- 提供网络安全保障，防止被监控或限制

## 功能

### 1. **绕过 DNS 劫持**
   - 配置 OpenWRT 路由器使用自定义的 DNS 服务，以绕过校园网中的 DNS 劫持。
   - 设置 DNS-over-HTTPS (DoH) 或 DNS-over-TLS (DoT) 来保护你的网络隐私。

### 2. **解除访问限制**
   - 介绍如何通过设置 VPN、代理等方式来解除对部分网站或服务的访问限制。
   - 提供一些常见的绕过技术，如 Shadowsocks、V2Ray、OpenVPN 等的配置方法。

### 3. **优化带宽和稳定性**
   - 设置 QoS（服务质量）规则来优化网络带宽，确保关键应用（如视频会议、学习软件等）获得优先权。
   - 配置中继功能，提升校园网无线信号的覆盖范围和质量。

### 4. **提高网络安全**
   - 配置防火墙和入侵检测系统，增强校园网环境下的安全性。
   - 使用 OpenWRT 提供的加密服务，保护网络中的数据传输。

## 使用指南

### 1. **硬件要求**
   - 一台支持 OpenWRT 固件的路由器。推荐使用具有较好性能和稳定性的路由器，如 TP-Link、Netgear 或华硕等品牌的路由器。

### 2. **安装 OpenWRT**
   - 请访问 [OpenWRT 官网](https://openwrt.org/) 下载适合你路由器型号的固件，并按照官网的安装教程进行刷机。

### 3. **配置步骤**
   以下是基本配置步骤，可以根据你的具体需求进行调整：

   1. **更改 DNS 设置**：
      - 登录到 OpenWRT 路由器的管理界面，进入“网络” -> “接口”，修改 DNS 设置为公共 DNS，如 Google DNS（8.8.8.8、8.8.4.4）或 Cloudflare DNS（1.1.1.1）。

   2. **设置 VPN 连接**：
      - 安装并配置支持的 VPN 客户端，如 OpenVPN 或 WireGuard。详细教程请参考项目中的配置指南。

   3. **启用中继模式**：
      - 在“无线”设置中启用中继模式，将路由器配置为无线中继，扩展校园网的信号覆盖范围。

   4. **配置 QoS 优先级**：
      - 在“网络” -> “带宽管理”中设置带宽管理规则，保证重要应用（如学习网站、视频会议等）获得优先带宽。

   5. **增强安全性**：
      - 启用 OpenWRT 的防火墙规则，并设置 NAT 规则来保护你的网络免受潜在的校园网监控和攻击。

## 常见问题

### 1. **如何选择合适的 OpenWRT 路由器？**
   - 推荐选择具有较强硬件性能的路由器，如支持 5GHz Wi-Fi 的路由器，且拥有至少 128MB RAM 和较高的 CPU 性能。

### 2. **如何处理 VPN 连接不稳定？**
   - 确保使用稳定的 VPN 协议，并尝试更换不同的 VPN 服务器。
   - 如果可能，选择离校园较近的 VPN 服务器以减少延迟。

### 3. **是否有其他方案绕过校园网限制？**
   - 除了 VPN 和代理，您还可以尝试使用自定义的 DNS 或通过中继设置提高信号质量，具体方法详见项目中的教程部分。
